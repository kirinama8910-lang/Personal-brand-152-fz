# Project Starter Kit — Стартовый набор для проекта с Claude Code

## Как использовать

1. Скопируй содержимое этой папки в корень нового проекта
2. Удали этот README.md (он нужен только как инструкция к набору)
3. Заполни PROJECT_PASSPORT.md — это первое, что нужно сделать
4. В начале первой сессии с Claude Code отправь ему CLAUDE_CODE_RULES.md

## Что внутри

| Файл | Назначение | Когда заполнять |
|------|-----------|-----------------|
| `CLAUDE_CODE_RULES.md` | Правила работы для Claude Code | Готов сразу, адаптируй под проект при необходимости |
| `PROJECT_PASSPORT.md` | Паспорт проекта — «точка входа» для всех | В начале проекта, обновлять каждую сессию |
| `PATTERNS.md` | Реестр решений и паттернов | По ходу работы, когда появляются находки |
| `DECISIONS.md` | Журнал архитектурных решений | Когда выбираешь между вариантами |
| `CASES.md` | Журнал проблемных кейсов | Если в проекте есть что тестировать на реальных данных |

## Порядок первой сессии

```
1. Заполни PROJECT_PASSPORT.md
2. Отправь Claude Code: «Прочитай CLAUDE_CODE_RULES.md. Подтверди и перескажи ключевые правила.»
3. Сформулируй первую задачу
4. После сессии: обнови паспорт, запиши паттерны и решения, если были
```
