# 🌐 Личный лендинг Марины Кириной

> **«Создаю сайты, которые не штрафуют»**

Персональный сайт-лендинг веб-разработчика Марины Кириной — одностраничник, соответствующий требованиям 152-ФЗ на уровне UI.

---

## 🛠️ Стек технологий

| Технология | Описание |
|---|---|
| HTML5 | Семантическая разметка |
| CSS3 | Адаптивные стили, mobile-first |
| Vanilla JS | Без фреймворков и библиотек |

- **Один файл** — весь сайт в `index.html`
- **Без внешних зависимостей** — никаких CDN, npm-пакетов, внешних скриптов
- **Без Google** — нет Google Fonts, Google Analytics, Google Tag Manager

---

## ✅ Соответствие 152-ФЗ

Сайт спроектирован с учётом требований Федерального закона № 152-ФЗ «О персональных данных»:

- 🍪 **Cookie-баннер** с раздельными согласиями (необходимые / аналитические / маркетинговые)
- 🔗 **Ссылка на управление согласиями** в футере — пользователь может отозвать согласие в любой момент
- 🚫 **Нет внешних трекеров** — никакие данные не передаются третьим сторонам без явного согласия
- 📄 **Политика конфиденциальности** — документ присутствует в составе лендинга
