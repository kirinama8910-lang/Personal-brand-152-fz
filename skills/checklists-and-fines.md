## ФИНАЛЬНЫЙ ЧЕКЛИСТ

### Все типы проектов
- [ ] HTTPS и редирект с HTTP
- [ ] Запрещённые сервисы удалены (Google Analytics, GTM, Fonts, reCAPTCHA, Facebook Pixel...)
- [ ] Яндекс.Метрика — только после согласия на аналитику
- [ ] Каждая форма: чекбокс согласия (не предустановлен)
- [ ] Каждая форма: ссылки на Политику и Согласие
- [ ] Кнопка submit заблокирована до согласия
- [ ] Маркетинговые рассылки — отдельный чекбокс
- [ ] В форме только необходимые поля (принцип минимизации, ст. 5 ч. 5)
- [ ] Страница `/privacy-policy` опубликована, доступна без регистрации
- [ ] Страница `/consent` — отдельный документ (с 01.09.2025)
- [ ] Страница `/cookie-policy` создана
- [ ] Ссылки на все документы в футере каждой страницы
- [ ] В Политике заявлена локализация данных в РФ
- [ ] Хостинг в России
- [ ] Уведомление подано в РКН

### Дополнительно: сайты с cookie
- [ ] Cookie-баннер при первом визите
- [ ] Кнопка «Только необходимые» равнозначна «Принять все»
- [ ] Аналитические / маркетинговые cookie выключены по умолчанию
- [ ] Cookie-предпочтения сохраняются в localStorage

### Дополнительно: интернет-магазин
- [ ] Платёжные данные — через шлюз с токенизацией
- [ ] Адреса доставки — цель указана в Политике
- [ ] Рассылки — отдельный чекбокс при оформлении заказа

### Дополнительно: SaaS / личный кабинет
- [ ] Кнопка «Удалить аккаунт» (30 дней на исполнение)
- [ ] Выгрузка личных данных (10 рабочих дней)
- [ ] Для B2B-клиентов — договор поручения обработки ПДн

### Дополнительно: мобильное приложение
- [ ] Экран согласия при первом запуске (не предустановлен)
- [ ] AppMetrica вместо Firebase Analytics
- [ ] Ссылка на Политику в настройках приложения
- [ ] Кнопка «Удалить аккаунт» в настройках
- [ ] Заполнена анкета App Privacy (App Store) / Data safety (Google Play)
- [ ] Политика конфиденциальности доступна по публичному URL
