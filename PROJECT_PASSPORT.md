# PROJECT PASSPORT — Personal-brand-152-fz

**Последнее обновление:** 2026-04-11

---

## Суть проекта

Статический лендинг личного бренда Марины Кириной (услуги: сайты, Telegram-боты,
автоматизации). Ключевое отличие — каждый проект «из коробки» соответствует 152-ФЗ:
политика, согласия, уведомление в РКН, самохостинг шрифтов.

## Целевая аудитория

Предприниматели и владельцы малого бизнеса в РФ, которым нужен сайт/бот без риска
штрафов от Роскомнадзора (до 18 млн ₽ с 30 мая 2025). Боль — «сделали сайт, а
потом прилетела проверка».

## Ценностное предложение

Сайт/бот + полный комплект 152-ФЗ-документов и настроек без доплат и доработок.

---

## Технический стек

| Компонент    | Технология                                      |
|--------------|-------------------------------------------------|
| Frontend     | Чистый HTML + CSS (без фреймворков, без сборки) |
| JS           | Ванильный, встроен в index.html                 |
| Шрифты       | Самохостинг (локальные файлы, без CDN)          |
| Хостинг      | Nginx + HTTPS (по 152-ФЗ — на территории РФ)    |
| Форма        | HTML-форма с согласиями 152-ФЗ                  |

## Архитектура

Одностраничный статический сайт + 3 правовых документа:

```
index.html          — главная (nav, hero, услуги, форма, футер, cookie-banner)
privacy-policy.html — Политика обработки ПДн
consent.html        — Согласие на обработку ПДн
cookie-policy.html  — Политика использования cookie
```

Все ссылки между файлами — относительные (миграция с серверных путей выполнена).

## Известные особенности и ограничения

- **Правила работы жёсткие:** пошагово, не трогать лишнее, паспорт — только
  через согласование (см. CLAUDE_CODE_RULES.md).
- **Всё в одном файле:** CSS и JS встроены в index.html (~2213 строк).
  Поиск и правки — через grep/Read, не через сборщик.
- **152-ФЗ — основа, не надстройка:** любые правки UI должны сохранять
  compliance (cookie-баннер, согласия, самохостинг, относительные пути).
- **Плейсхолдеры контактов** (`@your_username`, `marina@example.com`) пока
  не заменены — перед публикацией обязательно.
- Исправленный баг от 2026-04-11: рендер 3 правовых ссылок в левом верхнем углу
  `index.html` (был вызван глобальным `nav { position: fixed; ... }`).
