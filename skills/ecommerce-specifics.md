## ФАЗА 5А: ИНТЕРНЕТ-МАГАЗИН (дополнительно)

### Платёжные данные
Никогда не храни полные номера карт. Используй платёжный шлюз с токенизацией (ЮKassa, CloudPayments).

```javascript
// Хранить только: payment.id, last4, cardType — НЕ полный номер
const payment = await yookassa.createPayment({ amount, currency: 'RUB' });
```

### Адрес доставки = ПДн
В форме заказа: чекбокс согласия (или ссылка на Политику при наличии учётной записи). В Политике: цель «Оформление и доставка заказов».

### Рассылки — отдельный чекбокс при оформлении заказа

```html
<label class="consent-label">
    <input type="checkbox" name="consent_newsletter">
    <!-- НЕ предустановлен, отдельно от основного согласия -->
    <span>Хочу получать персональные предложения и новости</span>
</label>
```
