## Суть проекта

Лендинг личного бренда Марины Кириной — продажа услуги по настройке AI-бота для малого бизнеса (эксперты, салоны красоты). Центральная механика: квиз из 5 вопросов считает потери бизнеса в рублях от необработанных заявок, выводит диапазон и предлагает оставить контакт для консультации.

## Целевая аудитория

Владельцы малого бизнеса (эксперты, Beauty-мастера), у которых каждое сообщение и запись проходят через них лично. Боль: бизнес физически ограничен присутствием владельца — клиенты пишут вечером, в выходные, не дожидаются ответа и уходят.

## Ценностное предложение

Бот отвечает вместо владельца 24/7: записывает, отвечает на вопросы, возвращает спящих клиентов. Три пакета: Эксперт (от 45 тыс. ₽), Студия (от 75 тыс. ₽), Поток (от 110 тыс. ₽). Фиксированная цена, без скрытых часов.

## Технический стек

| Компонент | Технология |
|---|---|
| Frontend | Vanilla HTML + CSS + ES Modules (без фреймворка) |
| Шрифты | Локальные: Geologica Variable, Onest Variable, PT Mono |
| Квиз | Vanilla JS, модульная архитектура (screens/*.js) |
| Аналитика | Яндекс.Метрика (ID пока не задан, METRIKA_ID = null) |
| Webhook | fetch → POST JSON (URL пока __PLACEHOLDER__) |
| Хостинг | Не развёрнуто (планируется) |
| Юр. документы | Статические HTML-страницы |

## Архитектура

```
Пользователь открывает index.html
    ↓
Читает лендинг (hero → pain → story → packages → trust → faq → cta-final)
    ↓
Нажимает CTA → скролл к #quiz
    ↓
quiz-embed.js → quiz.js управляет экранами (screens/*.js)
    Путь 1: кнопка Telegram → deeplink → бот → webhook (fire-and-forget)
    Путь 2: резервная форма (Имя+Email+Телефон) → webhook (await)
    ↓
webhook.js → POST JSON на WEBHOOK_URL
    ↓
Экран thanks.js — «спасибо»

Параллельно:
cta-form.js → форма в #cta-final → тот же WEBHOOK_URL (path: 'cta_final')
cookie-banner.js → управляет согласием, пишет в localStorage
```


> но fetch-запросы на webhook будут заблокированы CORS.

## Известные особенности и ограничения

- **quiz-embed.js, quiz.css, quiz.html — не трогать**: это точка входа embed-виджета, изменения сломают встраивание квиза на внешние страницы
- **Запрещённые слова в копирайте:** Instagram, Meta, «блокировка», «смерть платформы», «вайбкодинг», «дёшево», «заменим администратора» — см. CLAUDE.md
- **CSS-переменные:** использовать только те, что объявлены в `:root` в `main.css`; `var(--accent)`, `var(--paper)`, `var(--inverted)` — не существуют
- **Пути к ресурсам** — только относительные: `./assets/...`
- **CSP в index.html** — при подключении webhook-домена добавить его в `connect-src`
- **Коммиты** — сообщения только на русском языке
- **Центральная боль лендинга** — зависимость бизнеса от присутствия владельца (вечная боль), а не потеря внешних платформ (ситуативный контекст)
